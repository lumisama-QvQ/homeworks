实现如下菜单驱动的学生成绩管理系统：
(在菜单的首行内容前先输出一行文字，包含学号和姓名)
(1) 输入每个学生的学号、姓名和各科考试成绩；
(2) 计算每门课程的总分和平均分；
(3) 计算每个学生的总分和平均分；
(4) 按每个学生的总分由高到低排出名次表；
(5) 按每个学生的总分由低到高排出名次表；
(6) 按学号由小到大排出成绩表；
(7) 按姓名的字典顺序排出成绩表；
(8) 按学号查询学生排名及其考试成绩；
(9) 按姓名查询学生排名及其考试成绩；
(10) 按优秀(90~100分)、良好(80~89分)、中等(70~79分)、及格(60~69分)、不及格(0~59分)5个类别，对每门课程分别统计每个类别的人数以及所占的百分比；
(11) 输出每个学生的学号、姓名、各科考试成绩，以及每门课程的总分和平均分；
(12) 将每个学生的记录信息写入文件；
(13) 从文件中读出每个学生的记录信息并显示。

分析：
    学生和学科数量均不确定，使用锯齿数组，数组上为学生的指针
    学生上绑定学科数量，学科的指针，平均分，总分
    成绩在0~100分范围内
    单独设置一个学科列表用于计算平均分
    在一个学生输入完成后立即计算总分和平均分
    每次修改了学生学科后立即和单独维护的学科列表对比
    分数设置为-1为缺考
    增加延时函数增加动感
    交互预计：
        1.学生和科目的输入：
            - 默认的学科数量和学科名称
            - 输出当前默认学科名称
            - 学生名，学号，按默认顺序输入各科成绩
            - 在输入学生名时可键入设置字样，调整学科
                - 第一行学科
                - 第二行对应排序（0,1,2,3）
                - 指令：    
                    - 删除
                        - 输入排序
                    - 更改
                        - 输入排序
                        - 更改后的学科
                    - 添加
                        - 添加的学科
            -返回输入学生和分数
           键入exit退出输入
        2.计算每门课程的总分和平均分：
            - 计算完成
        3.计算每个学生的总分和平均分：
            - 计算完成
        4.按每个学生的总分由高到低排出名次表：
            - 学生： 总分
            ...
        ...
        8.按学号查询学生排名及其考试成绩:
            - 输入学号
            - 排名， 各科成绩
        ...
        10.按优秀(90~100分)、良好(80~89分)、中等(70~79分)、及格(60~69分)、不及格(0~59分)5个类别，对每门课程分别统计每个类别的人数以及所占的百分比;
            - 计算完成
        11.输出每个学生的学号、姓名、各科考试成绩，以及每门课程的总分和平均分：
            - 学生， 学号， 各科成绩
            ...
            - 学科名：总分：100, 平均分
        12.将每个学生的记录信息写入文件：
            - 已写入record.txt
        13.从文件中读出每个学生的记录信息并显示:
            - 输入文件位置
            - 输出内容/读取失败
        0.退出
    结构体：
        学生：
            姓名：char*
            学科：struct*
            平均分：float
            总分：float
            学科数量：int
            参加的学科数量: int
        学科：
            学科名：char*
            学科总分：int
            学科分数：float
        学科维护列表：
            学科名：char*
            已参与人数：int
            学科实际人数：int
            累计总分数： float
            分数统计：int[5]
		等级：
			优：int
			良：int
			正常：int
			及格：int
			差： int
    需要函数：
        主菜单打印(void)
        输入学科(**struct 学科)
        删除学科(**struct 学科，int 删除序号)
        替换学科(**struct 学科, int 替换序号, *struct 替换学科)
        添加学科(**struct 学科, *struct 添加的学科)
        添加学生(**struct 学生, *struct 学科, *char 姓名, int 学科数量, float 总分,int 实际参与)
        更新维护列表(**struct 维护列表, *struct 学生)
        排名系统(*struct 学生， 模式)
        成绩表系统(unsigned int 模式, *struct 学生)
        查询系统(*void 内容， *struct 学生，unsigned int 查询模式)
        输出学生数据(*struct 学生)
        输出学科数据(*struct 维护列表)
        写入文档(*struct 学生)
        读取文档并显示(char *文件位置)
        延时函数(int 毫秒)
    全局标签：
    	升降序
    		- 升序
    		- 降序
    	查询模式
    		- 按名字
    		- 按学号
